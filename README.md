# site-web-guide

## 项目简介

`site-web-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面主要记录与网站浏览、使用或学习相关的静态内容，不针对任何特定域名或具体网站。

本仓库旨在以简洁、可维护的方式保存若干独立的 HTML 文件，方便随时查阅或分享。

## 目录说明

```
site-web-guide/
├── index.html          # 首页/导航页（若存在）
├── pages/              # 存放各个独立 HTML 页面的子目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源文件（如 CSS、图片等）
    ├── style.css
    └── ...
```

- `pages/`：每个 HTML 文件对应一个独立的归档页面，文件命名尽量反映内容主题。
- `assets/`：存放页面共用的样式、图像或其他静态资源。

## 页面归档说明

- 所有 HTML 页面均为独立文件，彼此之间无依赖关系（除非明确引用公共资源）。
- 页面内容以静态信息为主，不包含动态数据加载或外部 API 调用。
- 每个页面均可直接通过浏览器打开，无需搭建服务器（部分页面可能依赖本地资源路径）。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 归档页面，或对现有页面进行修正。
- 提交的页面应保持内容清晰、格式规范，避免包含外部链接、跟踪代码或商业推广内容。
- 请确保新页面不侵犯他人权益，且不违反适用法律法规。
- 对于长期未维护或不再适用的页面，可能会被归档或移除。
